# OTT 플랫폼 사용 가이드

## 🚀 빠른 시작

### 1. 프로젝트 실행

```bash
cd OTT_kiro_make
npm install
npm run dev
```

브라우저에서 `http://localhost:3001` 접속

### 2. 테스트 계정으로 로그인

- **일반 회원**: test@example.com / password123
- **LG U+ 회원**: lgu@example.com / password123

## 📱 주요 기능 사용법

### 회원가입

1. 우측 상단 "회원가입" 클릭
2. 이메일, 닉네임, 비밀번호 입력
3. 선호 태그 최소 1개 선택 (추천 알고리즘에 사용)
4. LG U+ 회원 체크 시 자동 구독 혜택
5. "회원가입" 버튼 클릭

### 콘텐츠 시청

1. 홈페이지에서 원하는 콘텐츠 클릭
2. 비디오 플레이어에서 재생
3. 하단 컨트롤로 재생/일시정지, 볼륨, 배속 조절
4. 시청 중 자동으로 시청 이력 저장 (10초마다)
5. 다음에 다시 시청 시 이어보기 가능

### 찜하기

1. 콘텐츠 상세 페이지에서 "찜하기" 버튼 클릭
2. 마이페이지 > 찜한 콘텐츠에서 확인 가능
3. 다시 클릭하면 찜 해제

### 댓글 작성

1. 콘텐츠 상세 페이지 하단 댓글 섹션
2. 댓글 입력 후 전송 버튼 클릭
3. 작성한 댓글 즉시 표시

### 검색

1. 헤더의 검색창에 키워드 입력
2. Enter 키 또는 검색 아이콘 클릭
3. 제목, 설명, 태그에서 검색

### 마이페이지

1. 우측 상단 닉네임 클릭 > 마이페이지
2. **프로필 탭**: 정보 확인 및 수정
3. **찜한 콘텐츠 탭**: 북마크한 콘텐츠 목록
4. **시청 이력 탭**: 최근 시청한 콘텐츠 (진행률 표시)
5. **통계 탭**: 시청 통계 및 태그별 분석

### 구독하기

1. 헤더 > 구독하기 또는 제한된 콘텐츠 클릭 시 안내
2. 베이직/프리미엄 플랜 선택
3. "구독하기" 버튼 클릭 (모킹 결제)
4. 구독 완료 후 모든 기능 이용 가능

### 콘텐츠 업로드 (스튜디오)

1. 헤더 > 스튜디오 (로그인 필요)
2. 제목, 설명 입력
3. 비디오 파일 업로드 (1-5분 권장)
4. 썸네일 이미지 업로드 (선택)
5. 태그 선택 (최소 1개)
6. "업로드" 버튼 클릭

- 📢 광고 노출

- 🚫 광고 없음
- 🔄 다음 화 자동 재생

- 🎁 일반 구독 무료 제공
- 자동으로 모든 혜택 이용 가능

## 🎮 키보드 단축키

비디오 플레이어에서 사용 가능:

- **스페이스바**: 재생/일시정지
- **←/→**: 10초 뒤로/앞으로
- **↑/↓**: 볼륨 조절
- **F**: 전체화면
- **M**: 음소거

## 💡 팁

### 추천 콘텐츠 받기

- 회원가입 시 선호 태그를 정확히 선택하세요
- 마이페이지에서 선호 태그 수정 가능
- 많이 시청할수록 추천 정확도 향상

### 시청 이력 관리

- 시청 이력은 최근 3개월간 저장
- 90% 이상 시청 시 "완료" 처리
- 마이페이지에서 진행률 확인 가능

### 찜 목록 활용

- 나중에 볼 콘텐츠를 찜하기로 저장
- 찜 목록에서 전체 연속 재생 가능
- 삭제된 콘텐츠는 자동으로 표시

## 🔧 문제 해결

### 비디오가 재생되지 않아요

1. 인터넷 연결 확인
2. 브라우저 새로고침 (F5)
3. 다른 브라우저에서 시도
4. 구독 상태 확인 (오리지널 콘텐츠는 구독 필요)

### 로그인이 안 돼요

1. 이메일과 비밀번호 확인
2. 테스트 계정 사용: test@example.com / password123
3. 회원가입 후 다시 시도

### 배속 재생이 안 돼요

- 배속 재생은 구독 회원만 이용 가능합니다
- 구독 페이지에서 구독 신청

### 검색 결과가 없어요

- 다른 키워드로 시도
- 태그로 검색 (예: #액션, #드라마)
- 오타 확인
