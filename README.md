# OTT 플랫폼

관리자 업로드 기반의 OTT 플랫폼으로, 1~5분 분량의 짧은 러닝타임 콘텐츠를 제공하며 시청 이력·추천·개인 통계에 초점을 둔 서비스입니다.

## 🎬 주요 특징

- **시리즈 콘텐츠**: 에피소드별 재생목록과 독립 댓글 시스템
- **구독 기반 권한**: 미구독/일반/프리미엄 3단계 구독 시스템
- **HLS 스트리밍**: 적응형 비트레이트 스트리밍
- **개인화 추천**: 태그 기반 맞춤 콘텐츠 추천
- **시청 통계**: 태그별 시청 패턴 분석

## 기술 스택

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router v6
- **Video Player**: HLS.js
- **Icons**: Lucide React
- **State Management**: React Context API

## 구현된 기능

### ✅ 시리즈 콘텐츠 (NEW!)

- 에피소드별 재생목록 UI
- 각 에피소드마다 독립적인 제목, 설명, 썸네일
- 에피소드 선택 시 자동 재생 전환
- 에피소드별 조회수 및 러닝타임 표시
- 현재 재생 중인 에피소드 하이라이트
- **에피소드별 독립 댓글 시스템**
- 10개의 시리즈 콘텐츠 (각 3-7개 에피소드)

### ✅ 인증 시스템

- 이메일 회원가입/로그인
- 닉네임, 선호태그 필수 설정
- LG U+ 회원 자동 구독 혜택
- 유효성 검증 및 중복 체크
- 회원 탈퇴

### ✅ 구독 시스템

- **미구독**: 크리에이터 콘텐츠만 시청, 480p 제한, 배속 불가
- **일반 구독 (₩5,000/월)**: 모든 콘텐츠, 1080p, 배속 재생
- **프리미엄 (₩9,900/월)**: 4K UHD, 동시 시청 4대
- **LG U+ 회원**: 무료 일반 구독

### ✅ 콘텐츠 시스템

- 오리지널 콘텐츠 (관리자 업로드)
- 크리에이터 콘텐츠 (사용자 업로드)
- 시리즈 콘텐츠 (에피소드 구조)
- 실시간 인기 차트
- 태그 기반 추천
- 검색 기능

### ✅ 비디오 플레이어

- HLS 스트리밍
- 배속 재생 (0.5x ~ 2x)
- 화질 자동 조절
- 이어보기
- 10초 단위 시청 이력 저장

### ✅ 마이페이지

- 프로필 관리
- 찜한 콘텐츠
- 시청 이력 (진행률)
- 태그별 통계

### ✅ 상호작용

- 찜하기
- 댓글 (에피소드별)
- 시청 이력

## 시리즈 콘텐츠 사용법

1. 홈페이지에서 "시리즈" 배지가 있는 콘텐츠 클릭
2. 콘텐츠 상세 페이지 우측에 에피소드 목록 표시
3. 원하는 에피소드 클릭하여 재생
4. 각 에피소드마다 독립적인 댓글 작성 가능
5. 에피소드 전환 시 자동으로 해당 에피소드 댓글 로드
